# ⚾ Text Baseball Simulation Game

콘솔 기반 텍스트 야구 시뮬레이션 게임입니다.  
플레이어는 감독이 되어 선수 스탯을 설정하고 경기를 운영합니다.

## 주요 기능

- 9이닝 경기 진행
- 타자 / 투수 스탯 (1~10)
- 실제 주자 상태 (1루, 2루, 3루)
- 투수 체력 시스템 (체력 감소 → 성능 하락)
- 스탯 기반 타석 결과 계산

## 스탯 설명

### 타자
- Contact: 안타 확률
- Power: 장타 / 홈런 확률
- Speed: (확장 예정 – 도루, 주루)

### 투수
- Control: 제구력 (안타 억제)
- Stuff: 구위 (장타 억제)
- Stamina: 체력 (낮아질수록 성능 저하)
